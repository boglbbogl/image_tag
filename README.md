# 🗨️ Image Tag


## Features
- ImageTag
- TagContainer
- TagItem
- TagTooltipOptions

-----------

## Support Platforms

- Flutter Android
- Flutter iOS
- Flutter Web
- Flutter Desktop



## Usage

**_Add the package to pubspec.yaml_**

```
dependencies:
  image_tag: ^<latest-version>
```

**_After that import the package_**

```
import 'package:image_tag/image_tag.dart';
```
